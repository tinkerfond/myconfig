# myconfig
Vim,Zathura,LaTeX configuration files....

### Configurations for:
1. `vim`
2. neovim
3. `zathura`
4. `texlive`
5. `fusuma`
6. `i3`
7. `zsh`

### Dependencies:
`rsync` (for deploying)

### Installation:
Execute initcfg.sh in terminal.
`./initcfg.sh`

#### WARNING: Automatic installation with initcfg.sh will overrwrite existing configuration
